# Eco-Scan: AI Waste Classifier

## Overview
Eco-Scan is a Bring Your Own Project (BYOP) submission for the Fundamentals of AI and ML course. It is a lightweight Natural Language Processing (NLP) tool that helps students accurately classify everyday campus waste into **Organic**, **Recyclable**, or **Landfill** categories using a Multinomial Naive Bayes algorithm.

## The Problem
Improper waste segregation is a daily issue on campus and in hostels. Students often get confused about where to throw items like "greasy pizza boxes" or "biscuit wrappers," leading to contaminated recycling bins. This project aims to solve that using text-based AI classification.

## Features
* Simple, fast Command Line Interface (CLI).
* Built using `scikit-learn` for ML and `pandas` for data handling.
* Custom dataset tailored to common campus and hostel waste.
